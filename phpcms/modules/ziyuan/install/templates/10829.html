盈亏计算

● 计算公式:

净值=余额 + 浮动盈亏 = 可用保证金 + 占用保证金 + 冻结保证金

占用保证金 = 建仓数量 * 固定保证金

净盈亏 = 总盈利 — 手续费 —仓息

风险率 = 净值 / 占用保证金 * 100%

● 风险控制:

当风险率≤5０％时，将对客户所有持仓单进行平仓结算。
